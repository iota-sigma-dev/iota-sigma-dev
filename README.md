## 🚀 Despliegue Actual: OrderKill (B2B SaaS)
Arquitecto Core y PM de **OrderKill**, plataforma B2B SaaS diseñada para refactorizar la logística y finanzas del sector gastronómico a nivel corporativo.

**Specs Arquitectónicos:**
* **Aislamiento Multi-Tenant:** Row-Level Security (RLS) en PostgreSQL. La seguridad de los datos reside en el motor, mitigando vulnerabilidades lógicas en la capa de aplicación.
* **Zero-Trust Network:** Infraestructura "Dark Host" sobre VPS + Cloudflare Tunnels. Dualismo de red estricto.
* **Inteligencia Agéntica:** Flujos asíncronos (RAG, vector embeddings) para soporte proactivo (CoWorker) orquestado con "Human-in-the-Loop" (HITL).
* **Ingeniería Determinista:** Monorepo (Turborepo), Next.js App Router (Edge Middleware) y pipelines CI/CD con *Artifact Integrity Guards*.
